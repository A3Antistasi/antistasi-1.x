DeFxy analyses any of the 3 bis formats for fxy files.

  • version 1.00 (ofp)
  • version 1.01 (arma)
  • version 1.02 (late arma3 releases)

Syntax:

DeFxy [-PF[8]] AnyfileOrFolder[.fxy] [destininationFOLDER]

-P Don't Pause

-F    full listing to NameOfFxy.TXT in utf8

standard use is to simply scan an fxy file, or folder of fxy files. for errors

brief output is sent to the screen

Full Listing

the dos screen is not capable of showing Unicode chars in all their glory because it is simply a 256 byte character set.

A full listing can also be quite large, taking an inordinate amount of time if sent to the dos screen. For this reason -F is specified

Optional Output folder

Normally the txt file for a full decode is in the same folder as the fxy file.

you can optionally specify it to go somewhere else.

Note that you cannot direct it to a different filename. Don't ask, and rename the result yourself if it bothers you.
